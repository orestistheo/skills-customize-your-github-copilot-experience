# 📘 Assignment: Python Data Visualization

## 🎯 Objective

Learn how to create charts and graphs in Python using `matplotlib` or `plotly` for exploratory data analysis and presentation.

## 📝 Tasks

### 🛠️ Load and Inspect Data

#### Description

Write a script that loads a CSV file into a DataFrame and summarizes key fields to understand the data distribution.

#### Requirements
Completed program should:

- Load data from `data.csv` using `pandas`
- Print row count, column names, and first 5 rows
- Compute summary statistics (mean, median, min, max) for numeric columns

### 🛠️ Create Visualizations

#### Description

Produce 3 different plots to highlight insights in the dataset, using `matplotlib` or `plotly`.

#### Requirements
Completed program should:

- Create at least one line plot, one bar chart, and one scatter plot (or equivalent)
- Label axes, titles, and legends clearly
- Save the plots to image files (e.g., `plot1.png`, `plot2.png`, `plot3.png`) or show interactive charts in Plotly
- Include comments about what each chart reveals about the data
